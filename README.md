## Abstract
Included in Abstract.pdf file



## How to use
copy both LTree.cpp and ltree file to required folder.

To use LTree in your code

```cpp
#include<ltree>
```
Don't forget to use namespace ksb to access LTree.
For Example,

```cpp
ksb::LTree<int> my_ltree(100, 3);
```


## Testing:

Use main.cpp file

```bash
g++ main.cpp
./a.out
```
